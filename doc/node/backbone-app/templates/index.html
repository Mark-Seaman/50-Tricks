Content from the HTML template!

Root directory

  • run - start the web app
  • app.js - defines pages in site
  • package.json - defines packages required by backbone

Client UI

  • js/boot.js - defines the client dependencies
  • js/BackboneDemo.js - define views for application
  • js/views/index.js - handle text insertion of template content
  • templates/index.html - content to insert

Server UI

  • views/index.jade - structure of pages
